# Available presets

Under the `preset` option, you can pass any of the following:

## Bootstrap 5 (default)

```js
app.use(Vue3Mq, {
	preset: "bootstrap5",
});
```

| Name | Minimum width |
| ---- | ------------- |
| xs   | 0             |
| sm   | 576           |
| md   | 768           |
| lg   | 992           |
| xl   | 1200          |
| xxl  | 1400          |
